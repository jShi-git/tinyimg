# tinyimg
压缩图片的工具

## 安装

    npm install -g tinyimg

## 准备工作

使用此工具，需要先到tinypng官网申请API key [https://tinypng.com/developers](https://tinypng.com/developers).

## 使用方法

```
    tinyimg -h
```
